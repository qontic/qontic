# Klein Tunneling Notebook

This notebook combines two applets:

- `2d/` contains the copied WebGL2 two-dimensional Dirac Klein tunneling project.
- `3d/` contains the copied WebGPU three-dimensional Dirac Klein tunneling project.

Open `index.html` through a local static server. The notebook embeds the 2D applet
immediately and loads the heavier 3D applet on demand.
